# Changelog

## 0.1.0

Initial release.

- 6 components: Button, TextInput, FormField, Modal, Toast, Autocomplete
- 3 themes: Dark, Parchment, White
- Three-tier token architecture (primitive, semantic, component)
- Multi-select Autocomplete with keyboard navigation
- Rubber hose motion system with `prefers-reduced-motion` support
- AI-native tooling: `gremlin-ui-api.yaml`, Cursor rules, Claude agents
